Detailed Description

template<typename T, typename Lock>
class cubos::core::memory::WriteGuard< T, Lock >

Utility class to provide safe read-write access to an object using a lock.

This class was created because there are multiple parts of the code that need to provide access to objects in a thread-safe manner.

Usage example: using AssetMetaWrite = core::memory::WriteGuard<AssetMeta, std::unique_lock<std::shared_mutex>>;

Template Parameters
TThe type of the object to guard.
LockThe type of the lock to use.
