Detailed Description

Archive implementation which allows reading from data embedded in the application as if it was a regular file or directory. Meant to be used with the cubinhos embed tool.

This archive works by provide a global map of data, where embedded data can be registered with a given name. Then, to access that data, an instance of this archive should be created with the same name, and then mounted anywhere on the file system.

The embed tool works by by generating source files which define and register that data. By linking those files into your application, your making the data available to be accessed through an instance of this archive.
