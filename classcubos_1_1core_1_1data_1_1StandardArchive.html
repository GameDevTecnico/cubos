Detailed Description

Archive implementation which allows reading and writing into a file in the OS file system, using the standard library.

Can interface with both regular and directory files. If the archive is not read-only and the target file is not found on the OS file system, then it is created.

This implementation does not detect changes in the file system made without the File and FileSystem classes (see #263).
