void cubos::core::net::TcpSocket::setBlocking(bool blocking)

Control blocking mode of inner socket.

Parameters
blocking Whether to block or not.

Note

A blocking socket does not return control until it has sent (or received) some or all data specified for the operation

A non-blocking socket returns whatever is in the receive buffer and immediately continues.
