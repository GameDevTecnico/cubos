cubos::engine::LocalToWorld struct
#include <engine/transform/local_to_world.hpp>

Component which stores the transformation matrix of an entity, from local to world space.

Contents

  • Reference
    • Public functions
    • Public variables

Note

This component is written to by the transform plugin, thus it never makes sense to write to it manually.

See also

The value is calculated from the LocalToParent component and the parent's LocalToWorld, if there's any.
