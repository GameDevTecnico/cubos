Define documentation

#define CUBOS_REGISTER_COMPONENT(type, storageType, name)
#include <core/ecs/component/registry.hpp>

Macro used to register a component type as an alternative to calling cubos::core::ecs::Registry::add() manually.

Warning

Care must be taken to ensure that this macro is included from at least one translation unit (think of it as a .cpp file) in the executable, otherwise the component type may not be registered. E.g.: keeping it hidden in a .cpp file on a library may not work.
