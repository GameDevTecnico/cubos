tesseratos::Game class
#include <tesseratos/game/game.hpp>

Resource which holds the game's Cubos instance.

Contents

  • Reference
    • Public functions

The game may be in four different states:

  • Unloaded - Game plugin has not been loaded yet.
  • Loaded - Game plugin has been loaded.
  • Running - Game has been started, and is currently running.
  • Paused - Game has been started, but is currently paused.

void tesseratos::Game::unload()

void tesseratos::Game::load(cubos::core::ecs::Plugin plugin)

Parameters
plugin Game plugin.

void tesseratos::Game::reset()

If the game has been started, resets it to the loaded state.

Equivalent to calling load again with the currently loaded plugin.

void tesseratos::Game::start()

Runs the game's startup systems.

Aborts if the game if it has not been loaded, or if it has already been started.

void tesseratos::Game::update()

Runs the game's update systems.

If the game is paused, does nothing. Aborts if the game has not been started.

void tesseratos::Game::pause()

Pauses the game.

Aborts if the game has not been started. Does nothing if the game is already paused.

void tesseratos::Game::resume()

Resumes the game.

Aborts if the game has not been started. Does nothing if the game is not paused.
