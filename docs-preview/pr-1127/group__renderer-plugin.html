Engine » Renderer module

Creates and handles the lifecycle of a renderer.

See also

Take a look at the Renderer example for a demonstration of this plugin.

Renders all entities with the RenderableGrid component, using as cameras entities with the Camera component selected by the ActiveCameras resource. Lights are rendered using entities with SpotLight, DirectionalLight or PointLight components.

Note

Entities with the above entities will be ignored if they do not possess LocalToWorld components.

The rendering environment, such as the ambient lighting and sky color, can be set through the resource RendererEnvironment.

Settings

  • cubos.renderer.ssao.enabled - whether SSAO is enabled.
  • cubos.renderer.bloom.enabled - whether bloom is enabled.

Resources

  • Renderer - handle to the renderer.
  • RendererFrame - holds the current frame information.
  • RendererEnvironment - holds the environment information (ambient light, sky gradient).
  • ActiveCameras - holds the entities which represents the active cameras.
  • ActiveVoxelPalette - holds an asset handle to the currently active palette.

Components

  • RenderableGrid - a grid to be rendered.
  • Camera - holds camera information.
  • Viewport - holds viewport information.
  • SpotLight - emits a spot light.
  • DirectionalLight - emits a directional light.
  • PointLight - emits a point light.
