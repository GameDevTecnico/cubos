# Cubos {#mainpage}

## Architecture {#section_architecture}

Cubos is split into two libraries and one executable:
- \ref core - a library with utilities which is the base of both the `engine` and
`cubinhos`;
- \ref engine - a library which contains functionality needed at run-time,
specific to the `engine`;
- \ref cubinhos - an executable which provides the necessary tooling for
developing a game with the `engine`.
