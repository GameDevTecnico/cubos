Detailed Description

template<typename... TArgs>
class cubos::Event< TArgs >

Class used to easily implement the observer pattern. Example usage: 

Event<const char*> event;
event.registerCallback([](const char* msg) => {
    std::cout << msg << std::endl;
});
event.fire("Hello!"); // Prints "Hello!"

TArgsThe argument types of the callback functions.
