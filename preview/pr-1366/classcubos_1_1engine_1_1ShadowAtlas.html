cubos::engine::ShadowAtlas class
#include <engine/render/shadow_atlas/shadow_atlas.hpp>

Resource which stores shadow map atlases, large textures that hold the shadow maps for each shadow caster in a quadtree structure, reducing texture switching. There is an atlas for spot lights, and another for point lights.

Public variables

glm::uvec2 configSpotAtlasSize
Configured size of the spot shadow atlas texture, in pixels. Use this to change the resolution of the spot atlas. Note that the texture isn't immediately resized; use getSpotAtlasSize() to get the actual texture size.
glm::uvec2 configPointAtlasSize
Configured size of the point shadow atlas texture, in pixels. Use this to change the resolution of the point atlas. Note that the texture isn't immediately resized; use getPointAtlasSize() to get the actual texture size.

Whether the shadow atlas textures have already been cleared this frame.
