User Interface

The namespace cubos::core::ui provides UI related utilities. CUBOS. UI support is built on top of Dear ImGui.

ImGui integration

To start using ImGui on a CUBOS. window, you must first call the initialize function, passing the window handle. This will initialize ImGui and create the necessary resources.

At the end of the program, you should call the terminate function to release the resources.

For each frame, you must call the beginFrame function before any ImGui calls, and the endFrame function after all ImGui calls, to render the UI.

Showing objects on the UI

The show function can be used to show any serializable object on the UI. It will show the object's properties on a ImGui table. There's also the edit function, which is similar to show, but allows editing the object's properties, in case it is deserializable.
