Detailed Description

template<typename T>
class cubos::engine::JSONBridge< T >

A bridge for loading and saving assets which are serialized to and from a JSON file.

This bridge automatically serializes and deserializes assets of type T to and from a JSON file. Thus, T must be serializable and deserializable. No additional context is given to the serializer or deserializer.

Template Parameters
TThe type of asset to load and save. Must be default constructible.
