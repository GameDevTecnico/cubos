Detailed Description

Abstract render device type, used to wrap low-level rendering APIs such as OpenGL.

See also
io::Window, gl::OGLRenderDevice.

◆ memoryBarrier()

virtual void memoryBarrier ( MemoryBarriers  barriers)
pure virtual

Defines a barrier ordering memory transactions. This ensure that all memory transactions before the barrier are completed before the barrier is executed. This function isn't supported on some platforms. Support can be queried using renderDevice.getProperty(Property::ComputeSupported).

Parameters
barriersThe barriers to apply.
