Detailed Description

template<typename T, typename Lock>
class cubos::core::memory::ReadGuard< T, Lock >

Utility class to provide safe read-only access to an object using a lock.

This class was created because there are multiple parts of the code that need to provide access to objects in a thread-safe manner.

Usage example: using AssetMetaRead = core::memory::ReadGuard<AssetMeta, std::shared_lock<std::shared_mutex>>;

Template Parameters
TThe type of the object to guard.
LockThe type of the lock to use.
