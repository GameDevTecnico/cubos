Detailed Description

template<typename T, unsigned int M = DEFAULT_FILTER_MASK>
class cubos::core::ecs::EventReader< T, M >

Used to read events, filtering is also possible via M parameter. In case M template parameter is not provided, it will read all events (no filtering).
